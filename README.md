CNN Pre-trained Model for MNIST Digits
This project demonstrates the use of a pre-trained Convolutional Neural Network (CNN) to classify handwritten digits from the MNIST dataset.

Dataset: MNIST (28x28 grayscale images of digits 0-9)

Model: Pre-trained CNN for digit recognition


Functionality:
Load the MNIST dataset
Preprocess the images for the CNN
Use the pre-trained CNN to predict digits
Evaluate the model accuracy on the test set
